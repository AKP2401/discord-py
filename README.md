# discord-py-bot
 Basic Discord Bot with python

## Features
* Hello
* Ping
* Permission Checker
* Ban/Unban
* Kick
* Stats for :
  * DOTA 2
  * CSGO
  * R6 Siege (currently not working)
  * Fortnite
* COVID Tracker
* GIF search (has randomness)
* Random String Creation

Hosted in Heroku
